# FishClicker

FishClock is a simple and practical Android app designed for anglers.

The app combines a clock, stopwatch, and timer with an easy-to-use net counter.
It helps you keep track of time and catches during a fishing session without unnecessary complexity.

## Features

- Live clock display
- Stopwatch mode
- Timer mode
- Net counter (multiple nets)
- Simple and clear user interface
- Works offline
- No accounts, no ads, no tracking

## Usage

- Swipe left or right in the stopwatch/timer area to switch between stopwatch and timer
- Start and stop the timer with one tap
- Use the net counter to track catches per net
- Long press on the top area to change the accent color

FishClicker is designed to be fast, reliable, and easy to read even outdoors.

## Privacy

FishClicker does not collect, store, or share any personal data.
The app works completely offline and does not use analytics or third-party services.

## Platform

- Android
- Built with Kotlin and Jetpack Compose
